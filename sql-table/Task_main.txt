Создать таблицу, бд

Вывод товаров на экран
1. Номер товара 2. ID товара 3. Название 4. Количество 5. Цена

Кнопки:
1. Добавить товар 2. Изменить товар 3. Удалить товар

Добавить товар -> новая страница с полями ввода: Название, Количество, Цена, Кнопка добавить (возвращающая на основную страницу)
Изменить товар -> новая страница с таблицей, где id - ссылка
Удалить товар -> новая страница с таблицей, где id - checkbox
